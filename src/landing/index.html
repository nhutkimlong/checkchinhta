AI Check Chính Tả

Công cụ kiểm tra chính tả tiếng Việt thông minh cho Microsoft Word.

Bước 1: Tải bộ cài đặt

Tải file cài đặt tự động về máy tính của bạn.

⬇️ Tải Installer (.cmd)
Bước 2: Chạy cài đặt

1. Nhấn đúp vào file installer.cmd vừa tải.

2. Nếu Windows hỏi xác nhận, chọn Run anyway (Chạy bằng mọi giá).

3. Chờ thông báo "CAI DAT THANH CONG".

Bước 3: Mở trong Word

1. Mở Word và chọn tab Insert (Chèn).

2. Chọn Add-ins (hoặc "Get Add-ins").

3. Chọn tab SHARED FOLDER (Thư mục chia sẻ).

(Lưu ý: Đây là tên mục trong Word, Add-in đã được cài vào máy của bạn)

4. Chọn AI Check Chính Tả và nhấn Add.

*Lưu ý: Nếu gặp lỗi bảo mật khi chạy script, hãy mở PowerShell với quyền Admin và chạy lệnh: Set-ExecutionPolicy RemoteSigned
